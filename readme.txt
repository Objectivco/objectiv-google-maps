The easiest way to create a searchable google map with your custom post types.

== Description ==

Objectiv Google Maps is the easiest way to create a searchable google map for your site. Objectiv Google Maps will provide an intuitive and easy way to create a list of locations that will be displayed as markers on a Google map. The map will be searchable by city with Google's autocomplete library.

**Features**

- Select from a list of all post types already defined on your site
- Add an address custom field to the selected post type with Google's autocomplete library to ensure that address are accurate
- Display info windows for each marker on the map that includes post title and address
- Search for locations by city with Google's autocomplete library
- Customize the look of the map with simple settings
- Displaying a map is as simple as adding [objectiv_google_maps] to a page or post

== Usage ==

Upload the plugin and customize your map in "Settings > Google Maps" to get started. Once you have selected a post type, each post will have a custom field for you to input an address for the post. This is the address that will be used to place a marker on the map.

To display the map, all you have to do is add [objectiv_google_maps] to a post or page. Everything else will be taken care of.

== Installation ==

Installing "Objectiv Google Maps" can be done by searching for "Objectiv Google Maps" on the "Plugins > Add New" page in your WordPress admin, or by following the steps below.

1. Download the plugin via WordPress.org
1. Upload the ZIP file through "Plugins > Add New > Upload"
1. Activate the plugin on the "Plugins" menu page
1. Edit plugin settings on the "Settings > Google Maps" page
